Centpercent: A coding challenge

A javascript coding exercise using loops. Write a program that displays the numbers from 0-100. Display the even numbers in BOLD

Instrucitons

Enter a starting and ending value. Press the button and that entire range will appear. Decimals or numbers that are out of range are invalid and hence the code will not run if you decide to enter such values.
